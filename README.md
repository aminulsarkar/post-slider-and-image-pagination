# Post Slider with Image Pagination

A free WordPress plugin to create **post sliders with featured image–based pagination** — a premium-style feature made accessible to everyone. Fully responsive, secure, and shortcode-based.

---

## Features

- **Featured image pagination** for navigation (like premium themes)
- **Two pagination layouts**:
  - Vertical thumbnails on the **right** of the slider
  - Horizontal thumbnails **below** the slider
- Shows **post title, date, categories, excerpt, and Read More link**
- Featured images used as slider background
- Fully responsive and lightweight
- Translation-ready
- Shortcode-based — no admin dashboard required (yet)

---

## Shortcodes

### Vertical Slider

[recent_posts_swiper_vertical posts="5" category="news,events"]


### Horizontal Slider

[recent_posts_swiper_horizontal posts="5" category="news,events"]


**Attributes:**

| Attribute | Description |
|-----------|------------|
| posts     | Number of posts to display |
| category  | Category slug (single or comma-separated) |

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin via the WordPress dashboard
3. Insert the shortcode into any post, page, or builder
