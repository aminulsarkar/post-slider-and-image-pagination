A free WordPress plugin to create post sliders with **featured image–based pagination**, just like premium themes. Fully responsive, shortcode-based, and lightweight.

== Description ==

Post Slider with Image Pagination lets you create modern post sliders where the navigation uses **featured images instead of dots or numbers**.  

This plugin is easy to use with shortcodes and fully responsive. Admin settings and more post type support will be added in future updates.

== Features ==

* Featured image–based slider navigation (premium-style)
* Two pagination layouts:
  * Vertical thumbnails (right side)
  * Horizontal thumbnails (below slider)
* Displays post title, date, categories, excerpt, and Read More link
* Featured image used as slide background
* Fully responsive on all devices
* Translation-ready
* Lightweight and secure
* Shortcode-based — no admin panel required yet

== Shortcodes ==

Vertical slider:
[recent_posts_swiper_vertical posts="5" category="news,events"]

Horizontal slider:
[recent_posts_swiper_horizontal posts="5" category="news,events"]

Attributes:

* posts — Number of posts to display
* category — Category slug (single or comma-separated)

== Supported Post Types ==

* Posts (currently)
* Pages and custom post types will be added in future releases

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress dashboard
3. Insert the shortcode into any post, page, or builder

== FAQ ==

= Does this plugin have an admin settings page? =
No, currently the plugin is shortcode-only. Admin settings will be added in future updates.

= Does it support custom post types? =
Not yet, but support is planned in future releases.

= Is it responsive? =
Yes, it works on desktops, tablets, and mobiles.
